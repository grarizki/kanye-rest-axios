# What I Learn

Frontend environment for building JavaScript applications

- Babel Loader
- Sass Loader With Dart Sass
- HTMLWebpackPlugin
- WebpackDevServer
- Asset Resource Loader
- Source Maps
- Caching
- Axios

## Usage

Run Dev Server (Port 3000)

```
  npm run dev
```

## Build for production

```
npm run build
```

## Kanye Quotes App

- index.js (main javascript)
- template.html (template)
